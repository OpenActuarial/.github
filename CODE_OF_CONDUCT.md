# Code of Conduct

Be professional and be kind. Specifically:

- Critique numbers, methods, and code — never people.
- No harassment, discrimination, or personal attacks, in any project
  space (issues, PRs, discussions, commits).
- Disagreements about actuarial methodology are welcome and are settled
  with references and reproducible examples, not volume.
- Assume good faith; write as if a regulator will read it someday.

Report unacceptable behavior privately via GitHub to the maintainer
(@OpenActuarial owner). Consequences range from a warning to a ban at the
maintainer's discretion.
